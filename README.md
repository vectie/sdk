# noetix_sdk_e1
Noetix robot E1 sdk.

### Prebuild environment
* OS  (Ubuntu 20.04 LTS or Ubuntu 22.04 LTS)  
* CPU  (aarch64 or x86_64)   
* Compiler  (gcc version 9.4.0) 

### Environment Setup

Before building or running the SDK, you can read the documentation linked here to ensure the following dependencies are installed:

[E1 SDK Environment Setup](https://noetixrobotics.feishu.cn/docx/PP2BdXkSUoQ2VOxctJKcEdHanAg?from=from_copylink).

### Build examples

To build the examples inside this repository:

```bash
./build_release.sh
```

### Notice
For more reference information, please go to [E1 DDS_SDK](https://noetixrobotics.feishu.cn/docx/YOZRd3ba3oetDHx9UWncuql0nzb?from=from_copylink).
